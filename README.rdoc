<h3>Explaination</h3>

<p>This blog app allows for simple CRUD of blogs for users to create their own blogs at a sub domain. Via afforementioned GEMS this editor has full wysiwyg text editor that allows for custom text, images, and videos.</p>

<h3>Usage</h3>

<p>For simple customization edit html.erb files to meet your needs. See the schema for column properties. For logged in user customization and linkage use <%= current_user %></p>
